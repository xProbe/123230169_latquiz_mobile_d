1. Login Page [20 Poin]
 Buat halaman login dengan 2 inputan (Username dan Password) (5 Poin)
 Buat error handling dan validasi username dan password dengan ketentuan
(10 Poin):
o Username dan password tidak boleh kosong (+4)
o Username yang valid: nama panggilan kalian (+2)
o Password yang valid: 3 digit terakhir NIM kalian (+2)
o Field di password bisa diatur visibilitasnya (on / off) (+2)
 Buat tombol yang bisa berpindah ke halaman Home Page jika username dan
password valid, sambal membawa data username (5 Poin)

 Menampilkan TextField search (10 Poin)
 * Search mampu berfungsi / berjalan sebagaimana mestinya untuk memfilter
list sesuai keinginan (5 Poin*)

Poin)
 Navigasi ke Detail Page jika salah satu card ditekan (5 Poin)

 Memiliki tombol back dan berfungsi (5 Poin)
